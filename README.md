# 🎮 LevelUp

### Where Learning Feels Like a Game

LevelUp is an AI-powered gamified learning platform that transforms study materials into interactive quests, challenges, and personalized learning adventures.

Students can upload handwritten notes, PDFs, or learning resources. Using AI, OCR, and OpenCV, LevelUp generates quizzes, flashcards, personalized learning paths, and an engaging quest-based experience.

## ✨ Key Features

* 📄 Smart Note Processing (OpenCV + OCR)
* 🤖 AI Tutor & Quiz Generation
* 🎮 XP, Levels, and Achievements
* 🐾 AI Study Companion
* 📊 Progress Tracking
* 🎯 Personalized Learning Paths

## 🛠️ Tech Stack

* Frontend: React, HTML, CSS, JavaScript
* Backend: Python (FastAPI / Flask)
* AI: Gemini API
* Computer Vision: OpenCV, Tesseract OCR
* Database: Firebase / Supabase

## 🎯 Vision

Make students forget they're studying and start enjoying the process of learning.
